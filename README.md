# kasztanlang
Język "programowania" z samymi kasztanami

Komendy:
```kasztanlang
kasztan+;
```
Zwiększ licznik o 1


```kasztanlang
kasztan-;
```
Zmniejsz licznik o 1


```kasztanlang
kasztan;
```
Wypisanie licznika


```kasztanlang
kasztan_jakisplik.php;
```
Uruchom plik php
